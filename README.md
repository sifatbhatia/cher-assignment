# Home Learning

Quick app I made to help people learn about home inspections.

## What it does

- Interactive lessons
- Simple quizzes 
- Progress tracking
- Coins (just for fun)

## Tech

- React + JS
- TailwindCSS 
- Context for state
- Vite

## Run it

```bash
npm install
npm run dev
```
